# assets/ — drop your images here

Put screenshots/figures in this folder, then tell Claude the filenames to wire them in.

## What to capture (priority order)
1. **effectgen.jpg** — Effectgen fitting shot. Ideally a single composite: garment image → on-model result (before→after). Highest visual payoff.
2. **homeliving.jpg** — Effectgen Homeliving: a furniture product placed into an interior scene.
3. **imagepiece.jpg** *(optional)* — a key figure from the AAAI paper.
4. **agentmon.jpg** *(optional)* — a screenshot of the generated/posted content or the Discord command.

## Specs (so the page stays fast)
- Format: **JPG** (photos) or **PNG** (figures/UI with text).
- Width: ~1200–1600px. Aspect ratio: keep them **consistent** (e.g. all 3:2 or all 16:9) for a clean grid.
- Size: compress to **under ~400KB each** (use https://squoosh.app or `sips`/ImageOptim).
- Make sure you have the rights to any model/product imagery shown.

## Optional: share-card image
One nice 1200×630 image named **og.jpg** can become the link-preview card (LinkedIn/Twitter). Effectgen before→after works well for this.
